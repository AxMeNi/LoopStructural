class LoopStructural.modelling.features.geological_feature.GeologicalFeature(name, interpolator, builder=None, data=None, region=None, type=None, faults=[], fold=None)¶

Bases: object

Geological feature is class that is used to represent a geometrical element in a geological model. For example foliations, fault planes, fold rotation angles etc.
